Glosario

Término Descripción Ejemplo
ICCID Es el número de la SIM. Contiene entre 19 a 20 caracteres. Este es un número único, el cual no puede repetirse entre validadores 8956030163614000276
MSISDN Es el número de teléfono asociado a la SIM. Cuando una sim es extraviada o robada, un nuevo ICCID es asignado al MSISDN. 56987787546
Nodo Concert IP Es la dirección IP a la que se conectará el validador. Para confirmar el MSISDN que tiene el validador, deberás hacer ping y encontrar este valor 10.228.56.1
Nodo Condor IP Es la segunda dirección IP a la que se conectará el validador. Es similar al Nodo Concert IP, pero varían su primeros números 10.228.56.1
